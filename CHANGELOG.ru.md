# История изменений (Changelog)

## v0.8.0

- **Обновления**: Проверка обновлений теперь дожидается завершения `git fetch` для каждого репозитория и корректно обрабатывает ошибки соединения (пропускает проверку при недоступности источника).
- **Процессы (Jobs)**: Убрано ограничение в 5 завершённых задач в панели процессов. Теперь отображаются результаты всех задач до их очистки пользователем.

## v0.7.0

- **Git Cache**: Устранена проблема блокировки `.git/index.lock` при параллельном обновлении нескольких skills из одного репозитория (очередь операций по ID источника).
- **Версионирование**: Улучшена стратегия разрешения версий и работы с кэшем без лишних сетевых запросов.
- **UI**: Добавлена детальная информация в карточке skill'а и исправлено отображение статусов.

## v0.6.1

- **UI/UX**: Мелкие исправления в интерфейсе, форме добавления источника и реестре.

## v0.6.0

- **Интерфейс**: Добавлен экран онбординга (Onboarding) и раздел Помощи (Help).
- **CLI**: Реализована проверка версии консольной утилиты `skills` и путей установки.
- **Локализация**: Расширены переводы (messages.ts) и исправлены тултипы.

## v0.5.0 и ранее

- Базовая реализация приложения: Source Manager, Version Resolver, Skill Registry, Installer Providers и Update Engine.
